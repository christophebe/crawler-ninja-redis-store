Widget Code

You can insert de following code into a markdown file to display a slider/carousel.


{ "name" : "carousel.jade",
  "properties" : {
    "id" : "slider1",
    "images" : [
      { "alt": "First slidexxxx",
        "src":"/images/carousel/xxx.png"
      },
      { "alt" :"Second slide",
        "src" : "/images/carousel/xxx.jpg"
      },
      { "alt" :"Third slide",
        "src" :"/images/carousel/xxx.jpg"
      }
    ]
  }

}
